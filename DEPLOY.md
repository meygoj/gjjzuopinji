# 🚀 部署指南

## 快速部署到Vercel（推荐！）

### 方法一：通过Vercel网站部署（最简单）

1. **登录Vercel** - 访问 [https://vercel.com](https://vercel.com)
2. **创建新项目** - 点击 "New Project"
3. **连接GitHub** - 将您的代码推送到GitHub仓库
4. **导入并部署** - Vercel会自动检测Next.js项目，一键部署！

### 方法二：使用Vercel CLI

```bash
# 1. 安装Vercel CLI
npm i -g vercel

# 2. 登录
vercel login

# 3. 部署
vercel

# 4. 部署到生产环境
vercel --prod
```
